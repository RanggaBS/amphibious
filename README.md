## Features

-   Speedtest
-   Uses decimal system format (KB, MB, GB, etc..) instead of binary system (KiB, MiB, GiB, etc..)
-   Redirect to WhatsApp when ordering in the voucher packages page
-   Formatted uptime (connected time) to hh:mm:ss
